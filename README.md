## Key bindings

### Tmux

`prefix` is Ctrl-A
`meta` is alt

`prefix + R`
- Reload tmux.conf

PS: If meta doesn't work on OSX, set it to ESC+

#### tmux-plugins

`prefix + I`
- Installs new plugins from GitHub or any other git repository
- Refreshes TMUX environment

`prefix + U`
- updates plugin(s)

`prefix + meta + u`
- updates plugin(s)
