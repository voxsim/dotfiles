# My dotfiles

## How to install it

```
git clone https://github.com/voxsim/dotfiles
cd dotfiles
git submodule update --init --recursive
pip3 install pyyaml
./install
```

## How to update it
```
git clone https://github.com/voxsim/dotfiles
cd dotfiles
git submodule update --init --recursive
./install
```
